# rss-feed-monitor (archived)

This project is no longer maintained. It fetched CBC's hourly news podcast
feed every 15 minutes via a GitHub Action and republished it as a static
RSS feed on GitHub Pages.

It has been superseded by a standalone Deno Deploy service that does the
same job more reliably and without relying on scheduled CI commits.

The scheduled GitHub Action has been disabled. The code here is kept for
historical reference only.
